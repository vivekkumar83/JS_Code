# variables:
* var & let used
* for constant const used
* loosely typed or dynamic language
```javascript
var myNum; // declare
myNum = 7; // assign
console.log(myNum);
myNum = 8; // update
console.log(myNum);

var newNum = myNum; // copy in another variable
console.log(newNum);

newNum = 9;
console.log(newNum);
console.log(myNum);

let firstName = "vivek";
firstName = "kumar";
console.log(firstName);

const pi  = 3.14; // cant update
```
